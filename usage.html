機能の使い方

このサイトはセキュリティ対策はほぼしていないので自己責任で使ってください。

簡単に説明しています。

sql

使う必要はないと思います。

phpに直書きしても動かない場合があります。

機能はそこまで豊富ではありません。

sql文をフォームに入れて送信を押すとphpに直書きできるコードが出力されます。

password

パスワードを自動生成します。

指定できるようになりました。

生成させたいのをクリックすれば生成されます。

passwd-cipher

パスワードを暗号化します。

パスワードをフォームに入れて送信を押すとパスワードが暗号化されたのが出力されます。

IP

自分のIPアドレスが確認できます。

vpnを使用していた場合はvpnのIPアドレスとなります。

random

フォームに書いた文がランダムに生成されます。

文字数の指定も可能です。

文字数の指定をしても指定した分の文字数が出ない場合もあります。

memo

セキュアではないので重要な情報を保存しないでください。

メモです。

メモの内容はcookieに保存されます。

cookieは通常、約3年間保存されます。

cookieを短い間のみ保存したい場合はチェックボックスに✅を入れてください。

入れた場合は約5分間のみ保存されます。

ファイル圧縮

重要なファイルをアップロードしないでください。

ファイルをダウンロードが終わったらすぐにサーバーからファイルを削除してください。

ファイルをzipかtar.gzに圧縮します。

手動でサーバーからファイルを削除できます。

ダウンロードされるファイル名はランダムです。

毎日0時に削除されてない圧縮ファイルは削除されます。

zipはパスワード付きにすることができますがパスワードは指定できません。

zipファイルのパスワードを忘れた場合は1からやり直してください。

アップロードできるのは

  • テキストファイル
  • 画像ファイル

    • のみです。

    zip,tar.gz以外の圧縮ファイルも対応予定です。
